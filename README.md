# PetShopApp

Приложение для управления зоомагазином (Pet Shop Management System).

## Описание
PetShopApp — это настольное приложение на Windows Forms (.NET 6), которое предоставляет функционал для просмотра каталога товаров, управления корзиной и заказами, а также администрирования базы данных (для менеджеров и администраторов).

## Функциональность
- **Гость**: Просмотр каталога, фильтрация, сортировка.
- **Пользователь**: Оформление заказов.
- **Менеджер**: Редактирование товаров.
- **Администратор**: Полное управление пользователями и данными.

## Требования
- .NET 6.0 Runtime (или SDK для сборки).
- Docker (для запуска базы данных).

## Запуск базы данных
База данных MS SQL Server разворачивается в Docker контейнере. 
Файл конфигурации `docker-compose.yaml` находится в директории `/opt/mssql` на сервере или может быть запущен локально.

```bash
docker-compose up -d
```

1. Перейдите в папку проекта:
   ```bash

   ```

   ```bash
   dotnet build
   ```
3. Запустите приложение:
   ```bash
   dotnet run
   ```
