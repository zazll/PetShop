# PetShopApp

Приложение для управления зоомагазином (Pet Shop Management System).

## Описание
PetShopApp — это настольное приложение на Windows Forms (.NET 6), которое предоставляет функционал для просмотра каталога товаров, управления корзиной и заказами, а также администрирования базы данных.

## Подключение к базе данных (ВАЖНО)

База данных находится на удаленном сервере, порт 1433 закрыт фаерволом провайдера.
Для подключения необходимо настроить **SSH-туннель**.

### 1. Настройка SSH-туннеля
Откройте терминал (PowerShell или CMD) и выполните команду:
```bash
ssh -L 1433:localhost:1433 root@94.156.170.23
```
Введите пароль (от сервера sfk).
Не закрывайте окно терминала, пока работаете с приложением.

### 2. Подключение через SSMS
- **Server name**: `localhost` (или `127.0.0.1`)
- **Authentication**: SQL Server Authentication

## Запуск приложения

1. Убедитесь, что туннель запущен.
2. Перейдите в папку проекта:
   ```bash
   cd PetShopApp
   ```
3. Запустите приложение:
   ```bash
   dotnet run
   ```
